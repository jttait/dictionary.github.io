Transport Layer Security (TLS)

TLS is a protocol that uses cryptography to secure communications over a network. The most widely known application is Hypertext Transfer Protocol Secure (HTTPS). TLS is the successor to the now-deprecated Secure Sockets Layer (SSL). The current version of the protocol is TLS 1.3 and previous versions are SSL 2.0, SSL 3.0, TLS 1.0, TLS 1.1, and TLS 1.2.
